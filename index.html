Projects

Categorical Feature Encoding Challenge II (Kaggle 34/1161) (2020)

In this competition, we used Pandas, Matplotlib to do data cleaning and removed abnormal rows, then conducted exploratory data analysis (EDA) on training data and created some features like combined features, frequency features, statistical features. We designed and tested blending and stacking method for model fusion based on XGBoost, LightGbm models to get final result.


Exploring the quality and physicochemical characteristics of Portuguese Vinho Verde wine (2020)

In this project, I used Imbalanced-Learn to deal with data imbalance problem, and conducted exploratory data analysis (EDA) on training data and use one-hot encoding processing in sequence classification problems. I designed and tested Voting Classifier models blending RF, Gradient Boosting Classifier and Extremely Randomized Trees models to improve the prediction accuracy of the model.
